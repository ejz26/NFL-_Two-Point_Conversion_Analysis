# 🏈 NFL Two-Point Conversion Data Analytics Project

## Overview

This project analyzes NFL two-point conversion play data to uncover trends in passing and rushing strategies, player involvement, and conversion success rates. The project demonstrates a complete data analytics workflow, from raw data processing to interactive business intelligence reporting.

The analysis was completed using **Python**, **PostgreSQL**, **SQL**, and **Power BI**, showcasing data cleaning, exploratory data analysis (EDA), database querying, dashboard development, and reporting.

---

## Project Objectives

- Clean and prepare raw NFL two-point conversion datasets
- Merge multiple datasets into a unified analytical dataset
- Perform exploratory data analysis (EDA)
- Store cleaned data in a PostgreSQL database
- Write SQL queries to answer key analytical questions
- Build an interactive Power BI dashboard
- Summarize findings in a professional report

## Project Workflow

### 1. Data Import & Cleaning
- Loaded multiple NFL datasets into Python using Pandas
- Removed unnecessary columns
- Handled missing values
- Merged passing, receiving, rushing, and defensive datasets
- Created new calculated columns to simplify analysis

### 2. Exploratory Data Analysis (EDA)
Performed exploratory analysis to better understand the dataset, including:

- Dataset dimensions
- Missing value analysis
- Distribution of passing vs. rushing attempts
- Success and failure frequencies
- Position distributions
- Receiver target analysis
- Summary statistics

---

### 3. PostgreSQL Database
- Created a PostgreSQL database
- Imported cleaned datasets from Python into PostgreSQL
- Organized data into analytical tables

---

### 4. SQL Analysis
Wrote SQL queries to answer business questions such as:

- Total passing and rushing attempts
- Passing vs. rushing success rates
- Most targeted receiver positions
- Receiver position success rates
- Most common rushing positions
- Rushing conversion success rates
- Player involvement in two-point conversions

---

### 5. Power BI Dashboard
Developed an interactive dashboard containing:

- Total two-point attempts
- Passing attempts
- Rushing attempts
- Passing success rate
- Rushing success rate
- Passing attempts by receiver position
- Rushing attempts by rushing position
- Success rates by position
- Target status distribution
---

### 6. Final Report
Created a report summarizing:

- Project objectives
- Methodology
- SQL findings
- Dashboard insights
- Key conclusions
- Recommendations
